package httplogger

import "go.astrophena.name/tools/internal/httplogger"

Package httplogger provides a http.RoundTripper middleware that logs HTTP requests and responses.

It wraps an existing http.RoundTripper and logs information about each request and response, including the start time, URL, method, status code (if available), and any errors. The logs are formatted with timestamps and indentation to visually represent the nesting of requests.
